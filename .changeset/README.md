# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works with multi-package repos to help you ship changes.

With multi-package repos, releases and changelogs become harder to manage. Changesets let contributors describe what changes they're making, and `pnpm changeset version` consumes those descriptions to bump versions and write changelogs.

## Common commands

```bash
# Add a new changeset (interactive)
pnpm changeset

# Bump versions + write changelogs based on accumulated changesets
pnpm changeset version

# Build + publish to npm
pnpm release
```

See https://github.com/changesets/changesets for full docs.
